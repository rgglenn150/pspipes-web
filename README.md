# pspipes-web

> Personal portfolio website showcasing mobile and web development projects

Built with [Next.js 16](https://nextjs.org), [React 19](https://react.dev), and [Tailwind CSS v4](https://tailwindcss.com).

![Next.js](https://img.shields.io/badge/Next.js-16-black?style=flat-square&logo=next.js)
![React](https://img.shields.io/badge/React-19-blue?style=flat-square&logo=react)
![TypeScript](https://img.shields.io/badge/TypeScript-5-blue?style=flat-square&logo=typescript)
![Tailwind CSS](https://img.shields.io/badge/Tailwind-v4-38bdf8?style=flat-square&logo=tailwind-css)

## Features

- Modern, responsive design with dark theme
- Animated gradient backgrounds and hover effects
- Optimized images with Next.js Image component
- SEO-ready with metadata configuration

## Tech Stack

| Technology | Purpose |
|------------|---------|
| **Next.js 16** | React framework with App Router |
| **React 19** | UI library |
| **TypeScript** | Type safety |
| **Tailwind CSS v4** | Utility-first CSS |
| **Geist Font** | Typography via next/font |
